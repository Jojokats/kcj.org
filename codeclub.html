CODE CLUB

Free, easy-to-start coding clubs for children aged 8-12.

Kids Code Jeunesse, in partnership with the Raspberry Pi Foundation, is supporting the growth of a nation-wide network of free volunteer-led coding clubs. We’re on a mission to build digital skills communities by encouraging volunteers and educators to bring coding opportunities to all Canadian children with Code Club Canada.

HOW DOES IT WORK?


An enthusiastic volunteer or educator teams up with a community venue, such as a school or library, to run an after-school club for young people aged 8-12, using specially created Code Club resources.


Code Club projects are easy to follow step-by-step guides, which help children learn Scratch, HTML & CSS and Python by making games, animations, and websites. The projects gradually introduce coding concepts to allow young people to build their knowledge incrementally, which also means there’s no need for the adult running the session to be a computing expert.


We also support active clubs with a range of extra resources, including certificates and posters, as well as offering competitions and prize draws for them.
