Inspirer et motiver les enfants canadiens avec le code

EN SAVOIR PLUS

Nous soutenons l'apprentissage de compétences numériques dans les classes partout au Canada en formant les enseignants, en développant du matériel didactique gratuit, en aidant les gouvernements à intégrer le code au programme scolaire, en offrant du matériel informatique et en encourageant les enseignants à créer des clubs de code gratuits. Nous travaillons également avec des chercheurs universitaires pour recueillir des données sur les enfants et les éducateurs qui apprennent à coder.

Nous créons des communautés de compétences numériques à travers le pays avec des ateliers, des festivals familiaux, des conférences et des Code Clubs. Vous pouvez nous aider à intégrer la programmation dans votre communauté en vous adressant à votre école primaire locale pour lui souligner l'importance d'apprendre à coder ou en créant un Code Club dans une école, une bibliothèque ou un centre communautaire.

Nous ne serions rien sans le soutien infaillible de nos bénévoles passionnés. Impliquez-vous en tant que mentor d'atelier, ambassadeur d'événement ou bénévole dans un Code Club pour nous aider à inspirer et aider des enfants à améliorer leurs compétences numériques à travers le pays. Nos bénévoles n'ont pas besoin d'être des programmeurs informatiques, nous avons aussi besoin d’artistes, de parents et d’enseignants de tous horizons, l'important est d’être passionné par l'avenir de nos enfants.

Code Club est un réseau de clubs de programmation gratuits dirigés par des bénévoles pour les enfants dans le cadre de Kids Code Jeunesse. Les Code Clubs font partie de notre mission de construire des communautés de compétences numériques dans tous le pays. N'importe qui peut créer un Code Club, que vous soyez un éducateur, un parent, un leader communautaire ou un bénévole!
