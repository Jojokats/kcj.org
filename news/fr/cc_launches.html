Cherchez-vous un moyen d’initier les enfants de votre communauté au codage? Que vous soyez bibliothécaire, enseignant, parent ou simplement passionné par l'alphabétisation numérique, voyez ce que Code Club Canada peut vous offrir. En partenariat avec Kids Code Jeunesse, les bénévoles et les éducateurs de Code Club Canada offrent des cours de codage gratuits en activités parascolaires et ce, dans toutes les provinces du Canada.

Code Club Canada compte plus de 135 clubs et ses projets sont traduits en 28 langues différentes. Des enfants de 8 à 12 ans ont donc la possibilité d'apprendre à coder avec Scratch, HTML et Python dans un environnement collaboratif et créatif. Grâce à ces ressources, les enfants peuvent créer leurs propres animations, des pages Web à l'aide de HTML ou utiliser Python pour développer leurs propres jeux. Vous pouvez accéder à ces projets au https://codeclubprojects.org/fr-FR/

Pour savoir comment devenir bénévole pour Code Club, organiser un événement ou démarrer un Code Club dans votre communauté, visitez notre nouveau site Web www.codeclub.ca et vous pourrez accéder à toutes nos ressources gratuitement. À titre de bénévole, vous pourrez non seulement enseigner aux élèves comment coder, mais vous jouerez également un rôle important afin de leur permettre de hausser leurs compétences en lecture, en écriture, en mathématique, en résolution de problèmes, en planification, en conception et en collaboration. De plus, nul besoin d’être un expert en codage pour faire du bénévolat!

Code Club Canada ouvrira officiellement ses portes à Montréal le 17 septembre 2017 à la Maison Notman. Lors du lancement, vous pourrez vous familiariser avec les projets, parler aux bénévoles et voir comment vous ou votre enfant pouvez vous impliquer dans un Code Club. L'événement est gratuit et ouvert à toute personne désirant aider des enfants de leur communauté à acquérir des compétences numériques grâce au code. Joignez-vous à nous!
