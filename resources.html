CODE, CREATE, EDUCATE!

We want to make sure every Canadian child is given the opportunity to learn computer programming. To do so, we help teachers and educators bring code into the classroom curriculum, into after-school programs, and into programs or Code Clubs at libraries and community centres.

Find below teaching material and learning resources we have made available for teachers and educators to start inspiring kids with code!
