RESOURCES

CODE, CREATE, EDUCATE!

We want to make sure every Canadian child is given the opportunity to learn computer programming. To do so, we help teachers and educators bring code into the classroom curriculum, into after-school programs, and into programs or Code Clubs at libraries and community centres.

Find below teaching material and learning resources we have made available for teachers and educators
to start inspiring kids with code!

Back python

    An introduction to the basic concepts of coding and to the Python programming language.

  • Python Lesson 2

    Students will learn about Python’s turtle, functions,comments, errors and common commands through discussion and practice.

  • Python Lesson 7

    Students will learn about while loops through discussion and practice.

  • Python Lesson 8

    Students will integrate game logic, user input, while loops and random integers to create a game of their choosing.

Back Unplugged

  • Introduction to HTML

    Students will learn the basic concepts of HTML (Hyper Text Markup Language).

  • Algorithm Activity 1: Ping-Pong Rescue

    Students will work in teams of 2 to 4 to create a simple algorithm that guides one of their team members, who is blindfolded, to rescue a ping-pong ball.

  • Algorithm Activity 2: Pizza Recipes

    Students will first engage in an interactive quiz in which they start to see how logic can be used to design complex algorithms. Afterwards, they will demonstrate their understanding by using logic to construct their favorite pizza recipes.

  • Algorithm Activity 3: Scavenger Hunt

    Students will continue to apply the concepts of complex algorithms by​ ​designing​ ​and​ ​participating​ ​in a scavenger hunt.

  • Choose Your Own Adventure

    Students will explore the various types of algorithms by designing and developing their own interactive Choose Your Own Adventure stories.

  • IIntroduction to CSS

    Students will learn the basic concepts of CSS (Cascading Style Sheet) and develop an understanding of the relationships between HTML markup and CSS declarations.

  • Unplugged User Stories

    Students will learn how to identify features and components of complex systems by constructing user stories.

  • Real-World Modelling with Wireframes

Back 2-hour Workshops

  • About a Canadian Animal

    Students will learn how to work with the block programming language, Scratch, and create a short project about an animal that they can then share with friends and family.

  • Drawing Logos

    After completing an introduction to drawing in Scratch, kids will be asked to imagine their own logos for 2067 and start a drawing project that they may complete in the workshop or continue at home.

  • Being Outer Space with Scratch

    The children will​ ​learn​ ​how​ ​to​ ​work​ ​with​ ​the​ ​block programming​ ​language,​ ​Scratch,​ ​ and​ ​create​ ​a​ ​short​ ​project​ ​about​ ​space​ ​that they​ ​can​ ​then​ ​share​ ​with​ ​friends​ ​and​ ​family.

Back Vibot the Robot Project

  • Scratch and Vibot lose their backdrop

    Students will learn to use dialogue, change backgrounds, and analyze situations as they change.

  • Scratch and Vibot, and the snowball attack

    Students will create a game in which Scratch has to run away from the snowballs thrown by Penguin. They are expected to create your own scoring system so that Scratch is able to make points

  • Scratch and Vibot have fun at Quebec City’s Winter Carnival

    In this challenge, Scratch and Vibot go to Quebec City’s Winter Carnival where there are several activities for them to do. The students must add necessary elements that will contribute to the success of the event.

  • Vibot and the Easter Scavenger Hunt

    In this challenge, Vibot goes on a scavenger hunt to find Easter eggs! It is up to the children to decide what will occur during the scene, and to help Vibot find the chocolates

  • Scratch and Vibot have lost their colours

    The children will help Scratch and Vibot to find their colours by identifying the coding block that can cancel the colour changes for both sprites. They will also add commands to make the sprites talk.

  • Vibot picks fruit

    The children have to identify all of the objects (sprites) that would be useful for you to illustrate the situation and to identify the links between them. The children are free to choose the shape that suits them.

  • Vibot and Scratch start a band

    The children will learn how to use instruments and the ‘sound’ commands in Scratch.

  • Scratch and Vibot escape the room

    The children will learn how to analyze the situation, to understand the situation, fix the code, and expand the program.

  • Vibot and Scratch go diving

    The children will reproduce the dialogue and movements as they are indicated.

  • Scratch, Vibot, and the seasons

    The children should represent each season within the animation with a background and a short story that is related to each one.

  • Vibot in London

    In this challenge, Vibot wants to visit London. To do this, the children have to help Vibot get on the bus and make it advance.

  • Scratch explains Christmas to Vibot

    Vibot and Scratch were playing in the snow. Suddenly, an object falls from the sky. Vibot thinks it’s an object but it’s actually Santa. Scratch tries to explain that it is Santa to Vibot. In this challenge, the children will continue the story by coding it in Scratch.

  • Vibot and Scratch help the Little Monkey

    The children will use ‘movement’ commands to create natural movements for the sprites.

  • Vibot and Scratch meet for the first time

    In this challenge, we invite the children to correct a text block that has crept into the Scratch programming blocks.

  • Vibot, Scratch, and Gobo play soccer

    The children will learn how to use the ‘movement’, ‘dialogue’, and ‘repetition’ blocks to make the sprites play soccer.

  • Vibot, Scratch, and Canada’s national sport

    In this challenge, Vibot, Scratch, and their friends play a game of table hockey. You are Team Blue and you control Scratch (WASD), Pico (arrows) and Nano, the goalie (space bar).

  • Scratch prepares a surprise for Vibot

    The kids will learn to make sprites appear and how to use the ‘broadcasting’ function for several different sprites.
