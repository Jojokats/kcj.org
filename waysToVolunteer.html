VOLUNTEER WITH US!

WAYS TO VOLUNTEER


Become an Event Ambassador

Become an Event Ambassador

Are you passionate about children learning to code? If you enjoy networking with community leaders at Meetups, tech conferences, gaming conventions, university fairs and special events then we would ask you to become a representative for Kids Code Jeunesse and Code Club Canada.

Mentor Kids at Workshops

Mentor Kids at Workshops

Do you enjoy the world of education and are looking for ways to give back to the community? Helping children to learn to code in the classroom is an extremely rewarding experience, especially when the “Aha, I get it!” moments occur – both from the children and the teachers. We’re looking for volunteers to support our instructors in the classrooms, at workshops, and at teacher training events.

Work Remotely

Work Remotely

Do you have skills to share and prefer working from home? We’d love your help with setting up events, fundraising, translation, sharing social media campaigns, publishing in WordPress, reviewing coding projects, testing volunteer training material and oh the list goes on.

Volunteer at a Code Club

CODE CLUB CANADA

Want to volunteer in your community? A project with Kids Code Jeunesse, Code Club Canada is a nationwide network of volunteer-led coding clubs for children aged 8-12 years old. Help us grow by volunteering at your local venue for an hour a week and help kids learn to make games, animations and websites, using Code Club Canada’s resources.

Contact your local library, community center or school to see if they’d like to start a Code Club and then register at: codeclub.ca
