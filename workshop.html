Code, Create, Teach

Workshops

The best learning involves lots of time for creativity and play! Our kids workshops let children explore code through art, music, games and story-telling. Our teacher training workshops give educators the tools they need to inspire kids with code.


Science Literacy Week

Type: Kids Workshop
Date: September 18 - 24, 2017 (dates to be confirmed by September 5th)
Location: Vancouver, Toronto, Ottawa, Kitchener Waterloo, Quebec City, Montreal

Science Literacy Week is a “week-long celebration of science” taking place across Canada this fall. As part of Science Literacy Week, Kids Code Jeunesse will be running free workshops in 6 Canadian cities to celebrate science in the context of literacy. These workshops will invite children to build a coding project based Chris Hadfield’s book The Darkest Dark.

Code Club Canada Launch

Type: Event
Date: September 17, 2017 (dates to be confirmed by September 5th)
Location: The Notman House, 51 Sherbrooke St W, Montreal, QC H2X 1X2

A project with Kids Code Jeunesse, Code Club Canada is a network of volunteers and educators who run free, extra-curricular coding clubs for children in local venues in every province across the country. We are officially opening our doors in Montreal on September 17th, 2017 at the Notman House. The launch will give you the chance to explore projects, talk to volunteers and see how you or your child can get involved in a Code Club. The event is free and open to anyone interested in helping kids in their community learn digital skills through code - join us!

App Inventor

Type: Kids Workshop
Date: November – December 2017 (dates to be confirmed by September 5th)
Location: The Notman House, 51 Sherbrooke St W, Montreal, QC H2X 1X2

This is a 5-Part workshop series on mobile application development for kids aged 8-12 who are familiar with Scratch. You will learn to code tactile interactive applications leveraging the unique capabilities of mobile devices such as photos, videos, position sensors, proximity, acceleration and orientation. These workshops continue to teach programming by introducing notions of parameters, date bases and distribution. This workshop is bilingual.
