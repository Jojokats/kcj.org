Code, Create, Teach

Workshops

The best learning involves lots of time for creativity and play! Our kids workshops let children explore code through art, music, games and story-telling. Our teacher training workshops give educators the tools they need to inspire kids with code.




Science Literacy Week - Edmonton

Type: Kids Workshop
Workshop date: September 23, 2017
Location: Edmonton - Rideau Park

Science Literacy Week is a “week-long celebration of science” taking place across Canada this fall. As part of Science Literacy Week, Kids Code Jeunesse will be running a free 2-hour workshop in Edmonton to celebrate science in the context of literacy. This workshops will invite children to build a coding project with Scratch based on Chris Hadfield’s book ‘The Darkest Dark’.

Intro to Hardware Bootcamp

Type: Kids Workshop
Workshop dates: September 23rd, September 30th, October 14th, October 21st, November 4th, and November 11th.
Location: Maison Notman House, 51 Sherbrooke St W, Montreal, QC H2X 1X2

It’s time to get hands-on! Kids Code Jeunesse is inviting kids to participate in a 6-Part hardware workshop, where they will get to experiment, code and play with Scratch, Raspberry Pi + Sense Hat, Makey Makey and Micro:bit! This workshop is offered in English (from 10am to 12pm) or French (from 1pm to 3pm).

Each child will receive their own Micro:bit and a copy of Margarida Romero's book, Vibot the Robot, an illustrated introduction to coding!

Scratch Bootcamp

Type: Kids Workshop
Workshop dates: September 24th, October 1st, October 15th, October 22nd, November 5th, November 12th.
Location: The Notman House, 51 Sherbrooke St W, Montreal, QC H2X 1X2

The Scratch Bootcamp is a 6-Part workshop where kids will learn to be creative, solve problems and code in a fun atmosphere. Kids will explore the basic notions of Scratch and will learn how to conceptualize their game ideas, master the concepts of games programming and know how to develop a game with advanced components.This workshop is offered in English (from 10am to 12pm)

App Inventor

Type: Kids Workshop
The workshop dates: November 12th, November 19th, November 26th, December 3rd, and December 10th.
Location: Maison Notman House, 51 Sherbrooke St W, Montreal, QC H2X 1X2

This is a 5-Part workshop series on mobile application development for kids aged 8-12 who are familiar with Scratch. You will learn to code tactile interactive applications leveraging the unique capabilities of mobile devices such as photos, videos, position sensors, proximity, acceleration and orientation. These workshops continue to teach programming by introducing notions of parameters, date bases and distribution. This workshop is bilingual.
