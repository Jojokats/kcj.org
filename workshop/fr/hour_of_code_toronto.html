“Une Heure de Code est une introduction d'une heure à l'informatique. Ce programme est conçu pour démystifier la programmation et montrer que n'importe qui peut en apprendre les rudiments. C’est maintenant un phénomène international de célébrer l’informatique avec des activités de programmation d’une heure mais aussi en se joignant à d’autres efforts de la communauté."

Faisant partie de l’édition Hour of Code de cette année, Kids Code Jeunesse offrira des ateliers de codage de deux heures à travers le Canada! Ces ateliers inviteront les enfants à créer un jeu simple avec le micro:bit, un ordinateur amusant, interactif et portatif qui vous permet d’être créatif avec le codage! Explorez le monde du codage et pratiquez votre pensée critique, votre résolution de problèmes et votre créativité avec nous!
